# Reforestation Impact Tracking

A Clarity smart contract for monitoring and rewarding tree planting initiatives with verifiable satellite data.

## Features

- Create reforestation projects with location data
- Satellite data verification system
- Automated reward distribution
- User reputation tracking
- Authorized verifier management

## Contract Functions

- `create-project`: Register new reforestation projects
- `submit-satellite-data`: Submit verification data
- `claim-rewards`: Claim STX rewards for verified trees
- `add-verifier`: Authorize satellite data verifiers
